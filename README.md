# Salarix

In order to run this demo you only must open the solution and start the Frontend and Backend project together in the following ports:

Frontend:
https://localhost:44392/

Employees Page
https://localhost:44392/employees

Backend:
http://localhost:59243/

Backend Endpoint:

All employees
http://localhost:59243/api/employees

One employees by its id
http://localhost:59243/api/employees/1
